Jupyter Notebook知识点

2019-03-09

Jupyter Notebook知识点
参考：
1.新建一个Notebook
点击界面右侧的new按钮，然后选择你想要启动的类型即可。
2.更改文件名字
在未更改之前界面最上方有初始文件名Untitled，点击即可更改文件名。
另一种方式：点击Fiel -> Rename。
3.notebook 界面组成
<1>notebook 的名称。
<2>主工具栏，提供了保存、导出、重载 notebook，以及重启内核等选项。
<3>快捷键。
<4>notebook 主要区域，包含了 notebook 的内容编辑区。
4.代码单元格（code cell）
可直接在单元格输入程序。
运行：<1>Shift + Enter。
<2>点击上方的运行按钮。
运行后会马上显示结果。
5.In[],out[]
In[]为程序
Out[]为返回的值（若没有返回值，直接打印则没有Out[]）
*(对应的程序和返回值[]里的数字相同)
6.设置标题单元格
选中第一个单元格，然后点击Insert -> Insert单元格above（在上方插入单元格）。
你会发现，文档的顶部马上就出现了一个新的单元格。
点击在快捷键栏中的单元格类型，将其变成一个标题单元格（heading cell）：
在快捷键区域有一个选择框。
选中下拉选项中的 Heading。
然后会出现一个弹出消息，告诉你如何创建不同层级的标题，这样你就有了一个不同类型的 cell：
7.不同层次的标题

   # : 一级标题
   ## : 二级标题
   ### : 三级标题
   ...
  （#越多，显示的字越小）
8.单元格操作
   *如果想删除某个单元格，可以选择该单元格，然后依次点击Edit -> Delete Cell；
   *如果想移动某个单元格，只需要依次点击Edit -> Move cell [up | down]；
   *如果想剪贴某个单元测，可以先点击Edit -> Cut Cell，然后在点击Edit -> Paste Cell [Above | Below]；
   *如果你的 notebook 中有很多单元格只需要执行一次，或者想一次性执行大段代码，那么可以选择合并这些单元格。点击Edit -> Merge Cell [Above | below]。
9.快捷键（参考：[](https://blog.csdn.net/lawme/article/details/51034543)）
  Jupyter Notebook 有两种键盘输入模式。编辑模式，允许你往单元中键入代码或文本；这时的单元框线是绿色的。
                                      命令模式，键盘输入运行程序命令；这时的单元框线是蓝色。
    <命令模式> (按键 Esc 开启)
       Enter : 转入编辑模式
       Shift-Enter : 运行本单元，选中下个单元
       Ctrl-Enter : 运行本单元
       Alt-Enter : 运行本单元，在其下插入新单元
       Y : 单元转入代码状态
       M :单元转入markdown状态
       R : 单元转入raw状态
       1 : 设定 1 级标题
       2 : 设定 2 级标题
       3 : 设定 3 级标题
       ...(以此类推)
       Up : 选中上方单元
       K : 选中上方单元
       Down : 选中下方单元
       J : 选中下方单元
       Shift-K : 扩大选中上方单元
       Shift-J : 扩大选中下方单元
       A : 在上方插入新单元
       B : 在下方插入新单元
       X : 剪切选中的单元
       C : 复制选中的单元
       Shift-V : 粘贴到上方单元
       V : 粘贴到下方单元
       Z : 恢复删除的最后一个单元
       D,D : 删除选中的单元
       Shift-M : 合并选中的单元
       Ctrl-S : 文件存盘
       S : 文件存盘
       H : 显示快捷键帮助
    <编辑模式> ( Enter 键启动)
       Tab : 代码补全或缩进
       Shift-Tab : 提示
       Ctrl-] : 缩进
       Ctrl-[ : 解除缩进
       Ctrl-A : 全选
       Ctrl-Z : 复原
       Ctrl-Home : 跳到单元开头
       Ctrl-Up : 跳到单元开头
       Ctrl-End : 跳到单元末尾
       Ctrl-Down : 跳到单元末尾
       Esc : 进入命令模式
       Ctrl-M : 进入命令模式
       Shift-Enter : 运行本单元，选中下一单元
       Ctrl-Enter : 运行本单元
       Alt-Enter : 运行本单元，在下面插入一单元
       Up : 光标上移或转入上一单元
       Down :光标下移或转入下一单元
10.保存
    快捷键第一个则为保存。
    保存后可在Home中找到。
