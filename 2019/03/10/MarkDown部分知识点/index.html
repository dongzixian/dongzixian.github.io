MarkDown部分知识点
参考：
1.标题
Markdown 支持两种标题的语法，类 Atx 和类 Setext 形式
(注意#后面有个空格)

         # 一级标题
         ## 二级标题
         ### 三级标题
   <Setext>(-与=数目任意，最好三个及三个以上)
        一级标题
        ======

        二级标题
        ------
2.粗体和斜体
    <粗体>
        **(要写的内容)**
        __(要写的内容)__
    <斜体>
        *(要写的内容)*
        _(要写的内容)_
3.换行
    <1>第一种写法
       直接敲两个回车键
    <2>第二种写法
       在写完一段后敲两个空格，然后回车写下一段
    <3>第三种写法
       在写完一段后用HTML的语法：<br />作为换行，然后写下一段
4.分割线
    可以在一行中用三个及以上的星号、减号、等于号、底线来建立分隔线，行内不能有除空格外的其他东西
    like:*******
         -------
         =======
         _______
5.引言
    <1>用一个">"就可以来写一个多行的引用
    <2>还可以每行用一个">"，这样比较美观
    <3>嵌套引用可用两个">"
6.无序列表
    无序列表可以在每行开头用星号、加号、减号来表示，也可以三者混合一起，推荐使用相同的字符，避免混乱。
    like:* 一本书     
         * 两本书
         * 三本书
         -------
         + 一个
         + 两个
         + 三个
         -------
         - 四个
         - 五个
         - 六个
    （最后前面符号显示为实心圆点）
7.有序列表
    有序列表用数字接着一个英文句点来表示，数字可无序
    like：
        1.第一    
        2.第二
        3.第三
        ------
        1.第一
        * 第二
        + 第三
        ------
        2.第一
        5.第二
        4.第三
    以上编辑的内容都是：
        1.第一
        2.第二
        3.第三
8.行内代码块
    一行中的代码块可用``(反引号)围住
    如果代码中含有(`)这个字符，则用`` ``两个反引号围住
    （这样代码就会有底色了）
9.链接
    网址链接有两种形式：行内式、参考式。
    不管是哪一种，链接文字都是用 [方括号] 来标记，双引号""的title可写可不写
    <行内式>
        [文字内容]（网址）
        [文字内容]（网址"title"）
        [文字内容]（网址'title'）
    <参考式>
        ·一般写法
        [文字内容][1]
        [1]：网址
        ·隐藏链接标记
        [文字内容][]
        [文字内容]：网址
        （这里的链接辨别标签可以有字母、数字、空白和标点符号，但是并不区分大小写）
10.图片链接
    图片链接与上面的网址链接类似，同样有两种形式：行内式和参考式，只不过图片链接在前面加上一个感叹号!
    <行内式>
        ![文字内容](图片链接)
    <参考式>
        ![谷歌][1]
        [1]:图片链接
